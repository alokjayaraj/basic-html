INTRODUCTION TO HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

HTML Tags

  • !DOCTYPE  :  Defines the document type.
  • html  :  Tag represents the root of an HTML tag.
  • head  :  The head element is a container for metadata (data about data) and is placed between the html tag and the body tag.
  • h1  :  Defines the most important heading.
  • h6  :  Defines the least important heading.
  • p  :   Defines a pararaph.
  • img  :  The img tag is used to embed an image in an HTML page.
  • em  :  The em tag is used to define emphasized text. The content inside is typically displayed in italic.
  • ul  :  The ul tag defines an unordered (bulleted) list.
  • ol  :  The ol tag defines an ordered list. An ordered list can be numerical or alphabetical.
  • li  :  The li tag defines a list item. The li tag is used inside ordered lists, unordered lists, and in menu lists.
  • div  :  The div tag defines a division or a section in an HTML document.The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
  • strong  :  The strong tag is used to define text with strong importance. The content inside is typically displayed in bold.
  • br  :  The br tag inserts a single line break.
